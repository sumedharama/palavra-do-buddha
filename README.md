**NOTE:**

The Ruby libs need the old Ruby v2.6.6.

``` shell
rvm pkg install openssl
rvm install ruby-2.6.6 --with-openssl-dir=$HOME/.rvm/usr
rvm use ruby-2.6.6 --default
gem install bundler:1.13.7
bundle install
bundle exec make epub
```
